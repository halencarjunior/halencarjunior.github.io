# AGENTS.md

This file provides guidance to Codex (Codex.ai/code) when working with code in this repository.

## Site Information
- Jekyll site using `jekyll-theme-hacker`, hosted on GitHub Pages at humbertojunior.com.br
- Author: Humberto Jr (bt0)
- All new posts must be written in English

## Post File Format
- Posts are `.html` files (not `.md`), with Jekyll front matter followed by HTML content
- Front matter fields: `layout: post`, `title`, `date`, `categories`, `tags`, `author`
- Date format: `YYYY-MM-DD HH:MM:SS -0300`
- Content is written in HTML (`<h2>`, `<p>`, `<pre>`, etc.), not Markdown
- No `_posts/` directory — posts live in category-based paths

## Directory Structure
Posts are stored at `category/YYYY/MM/DD/post-name.html`. The URL mirrors this path exactly.

Existing categories (directory names): `infosec`, `pentest`, `osint`, `bugbounty`, `writeup`, `tools`, `bfriday`, `criptografia`, `redes`, `ransonware`, `nodejs`

Multi-category posts combine dirs: e.g., `infosec/pentest/YYYY/MM/DD/post.html` with `categories: infosec pentest`

## Homepage (index.html)
- `index.html` is a fully static HTML file — **not** generated by Jekyll's post collection
- When creating a new post, manually add an entry at the top of the `<ul class="post-list">` in `index.html`
- Entry format:
  ```html
  <li>
    <span class="post-meta">Mon DD, YYYY</span>
    <h3>
      <a class="post-link" href="/category/YYYY/MM/DD/post-name.html">
        Post Title
      </a>
    </h3><p>Brief description here.</p>
  </li>
  ```
